## ▶️ How to Run This Project (Localhost)

### Prerequisites
- Node.js (v16 or higher)
- Any modern web browser

Check Node.js:
```bash
node -v
1. git clone https://github.com/makunno/demo_keylogger.git
cd demo_keylogger
2. Install dependencies

npm install


3. Start the server

node server.js


4. Open in browser

http://localhost:3000
